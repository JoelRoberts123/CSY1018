Alternative text - text that can be used to tell viewers the context of an image

Attribute - defines the property of a piece of code

CSS - Stands for Cacading Style Sheets. used to give web pages design

Body Tags - these contain the vast majority of content on your HTML pages

Browser - this is what viewers will view the website on

class - variable used in coding eg. h1, h2 etc

Closing tag - this denotes the end of a piece of content

Codepen - online code editing website, great for creating new code as well as searching for code that already exists

Containers - these help the CSS code certain peices of text

File path - refers to a specific location in a file system

Href - a reference to a link from another folder.

HTML - Hypertext markup language, used to construct webpages

img - refers to an image

Li - lists, used for menu's

Link - refers to a link to a webpage embedded in the coding

Navigation - refers to the way a user makes his way through the site, usually comes i nthe form of a navigation bar at the top of the page.

Opening tag - This denotes the start of a piece of content.

Root Directory - this is the folder that will contain all HTML and CSS files for a webpage

src - refers to the source of an image, can be a hyperlink or a link to an image located in a folder

Validator - this is used for checking HTML/CSS to make sure there are no errors
